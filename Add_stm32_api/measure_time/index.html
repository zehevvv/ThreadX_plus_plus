Measure time

This debug object that allow you to measure time in microseconds units.

Create

#include "Debug.h"
Debug::MeasureTime my_measure_time;

How to use

When you want to start measure you need to call to Start() function, and when you want to end call to End() function, the function End() return the delta in microseconds units,
Example: 

my_measure_time.Start();
SomeFunction();
uint32_t result = my_measure_time.End();
printf("The delta is %lu \n", result);

Note: You need to know that measure time is get is data from counter that work on 32 bit register, so it may cause to wraparound.
