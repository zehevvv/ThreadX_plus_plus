Install ThreadX on ST

Introduction

In this tutorial, we will install ThreadX by using the CUBE IDE, first, you need to install CUBE IDE and then open your project, after that, you can install ThreadX.
I write this tutorial but you do not must to use it, you can find a lot of tutorials on Google, just keep in mind that ThreadX needs to be config to use static memory allocation (I add an option to use also the dynamic memory allocation here)

Install CUBE IDE

You can download the CUBE IDE from here: Link
You also need to download CUBE MX: Link

Open project

  1. Open Cube IDE
  2. Go to File->New->STM32 project

  3. New window will open, in the field Commercial Part Number you need to add your STM32 model ("for example my is STM32H743ZIT6"),
    in the right bottom of the screen you will see a list of STM32 modules, select your model from the list, and on the bottom of the screen press on the Next >

  4. New window will open, in the field Targeted Language select C++, and in the field Project Name: Write the name of your project, and press the Finish button.

  5. New window will ask if you want memory Protection, use your brain and press Yes.

  6. Now wait, it can take some time, but in the end, you will get the next screen

Add ThreadX

  1. In the screen you need to select the tab Pinout & Configuration and select Software Packs, then select Manage software packs
  2. New window will open, select the tab STMicroelectronics, in this tab you need to find X-CUBE-AZRTOS-XX ("XX" is the name of your family model, for example, mine is "H7") then select it and list of packages will be open, you need to check the Azure RTOS STM32Cube expansion package for STM32XX series (again "XX" is the name of your family model), then press install
  3. After the download finish, Now you need to return to the main screen and again select the tab Pinout & configuration and select Software Packs, but now select Select Components
  4. New window will open, and you will see a list of all the packs you and add to your project, you need to select STMicroelectronics.X-CUBE-AZRTOS-XX(again "XX" is the name of your family model)-> RTOS ThreadX-> ThreadX, then select the checkbox of Core and press On "Ok" button , Note: If STMicroelectronics.X-CUBE-AZRTOS-XX is gray press on install, after the install select it.
  5. On the main screen on the left side you see a list of options, select Middleware and Software Packs, and it opens a list, then select X-CUBE-AZRTOS-XX, and it opens a new screen, check the box of field RTOS ThreadX ,
  6. Save the file (go to File->Save), and the IDE will generate the code, the first time will take some time.

Next step

Now you are ready to start with the real thing! Add ThreadX++.
